# comparadolar.ar

[ComparaDolar](https://comparadolar.ar/) es una plataforma que permite a los usuarios comparar las tasas de cambio del dólar en diferentes casas de cambio y bancos en Argentina. Soporta USD, USD CCL, USDC, USDT, BTC y ETH.
